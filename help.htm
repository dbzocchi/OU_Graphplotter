1. Changing the size of Graphplotter

To vary the size of Graphplotter, first click on the webpage outside Graphplotter or press F5 to reload Graphplotter. Then hold down the Ctrl key and either rotate the mouse wheel or press the + or - keys. This zooms the entire webpage in and out. To reset the page to its default size, click on the webpage outside Graphplotter or reload Graphplotter by pressing the F5 key, and then press Ctrl+0 (that is, press the Ctrl and 0 keys together). If necessary, use the vertical and horizontal scroll bars to move around the enlarged webpage.

2. Keyboard control

All of the Graphplotter components can be controlled by using the keyboard as an alternative to the mouse. To begin, you have to select Graphplotter, which you can do by clicking inside it. An alternative, which works in some browsers, is to repeatedly press the Tab key (this is called tabbing) until a component of Graphplotter is highlighted by a border around it.

A border like this around a Graphplotter component indicates that the component has keyboard focus (or focus for short). When a component has focus, you can control it with the keyboard. You can give a component focus in the following ways (some methods work for only some components).

  • Click on the component (e.g. an input box).
  • Use the shortcut keys given in the next section.
  • Tab through the components: each time you press the Tab key the focus is transferred to the next component in order. You can tab backwards by pressing the Tab key while holding down the Shift key (Shift+Tab).
  • For the graphing area, move the mouse pointer over this area.

When a component has focus, keyboard commands are available

  • When one of the tabs (e.g. One graph, Two graphs) has focus, pressing the Enter key activates that tab.
  • When a button has focus, pressing the Space key presses the button.
  • When an input box has focus, you can type characters into it. You must press the Enter key (while the box has focus) or change focus to another component for the change to take effect.
  • When a drop-down selector for choosing a function is in focus, pressing the Enter key or Space bar opens the drop-down list. Use the Up and Down arrow keys to navigate the list and then press the Enter key or Space bar to choose the highlighted option. The Home and End keys take you to the first and last functions in the list.
  • When a slider has focus, pressing the Left or Right arrow keys moves the slider left and right. If you hold down the Ctrl key while pressing an arrow key, then the slider moves faster. To move directly to the lower or upper limit of the slider, press the Home or End key, respectively.

To reload Graphplotter, make sure that the focus is on the web page and press the F5 key.

3. Shortcut keys

Some components have shortcut keys which, when pressed with the Ctrl and Alt keys held down, allow you to shift the focus directly to these components. To use the shortcut keys, you must have the focus on a Graphplotter component.

Ctrl+Alt+F Selects the Functions tab.
Ctrl+Alt+O Selects the Options tab.
Ctrl+Alt+1 or Ctrl+Alt+N Selects the One graph tab (which displays the graph of just one mathematical function).
Ctrl+Alt+2 or Ctrl+Alt+T Selects the Two graph tab (which displays the graphs of two mathematical functions).
Ctrl+Alt+H Selects the Help tab (which displays this help text).
Ctrl+Alt+G Selects the graphing area.

4. Choosing a function

Graphplotter can plot graphs of one or two functions. The functions are chosen by using the drop-down selectors at the top of the left-hand panel. You can choose to show one or two graphs by clicking on the appropriate tab at the top of the application (or using keyboard control as described in Section 2). The left- and right-hand selectors produce blue and green graphs, respectively.

To choose a predefined function, click on either of the drop-down selectors and then click on the desired function. The graph is plotted in the right-hand panel with the equation of the function displayed above. You can change the values of the constants – this is described in the next section.

To enter your own function, click on the Custom function option at the bottom of the drop-down list of functions. A Custom function input box appears at the bottom of the panel. You can type your own function into this box, using the syntax described in Section 11. (You can give the box focus by tabbing to it once you have selected Custom function.) After typing in a custom function, press the Enter key to plot its graph and display its equation.

When you enter a function in the custom function box, some information text appears in the text area at the bottom of the left-hand panel.

5. Choosing the values of the constants

When you choose a function, an input box and a slider is provided for each constant in the function. You can use either of these to change the value of a constant.

An additional checkbox control is provided for the functions y = abx+c and y = a logb(cx)+d. This allows you to easily set the constant b to be the mathematical constant e.

The sliders have lower and upper limits, which you can change if you wish. To change the lower limit to a lower value, type the new limit into the input box above the slider and press the Enter key. (This also sets the constant to that value.) If you want to make the lower limit higher than it already is, then type ‘min=’ followed by the new limit (e.g. ‘min=−10’) into the input box above the slider and press the Enter key.

Similarly, to make the upper limit higher, type the new limit into the input box above the slider and press the Enter key. If you want to make the upper limit lower than it already is, then type ‘max=’ followed by the new limit (e.g. ‘max=10’) into the input box above the slider and press the Enter key.

6. Changing the viewable area of the graph

You can change the viewable area of the displayed graph in any of the following ways.

  • Type new numbers into the x min, x max, y min and y max input boxes at the bottom right of the right-hand panel, and press the Enter key or select another component.
  • Use the arrow, zoom and ‘Autoscale’ buttons under the graph.
  • Move the graph with the mouse, by left-clicking on the graph and keeping the mouse button pressed while you move the mouse to drag the graph in the required direction. Release the mouse button to leave the graph in place.
  • Zoom in and out with the mouse, by clicking on the graph and rotating the mouse wheel.
  • Move the graph by using the keyboard. First, make sure that the graph has focus (indicated by a yellow box). To do this, move the mouse pointer over the graph, tab to the graph or press Ctrl+Alt+G. Then you can move the graph slowly by holding down the Shift key and pressing the Home and End keys for left and right, or the Page Up and Page Down keys for up and down. You can move it faster by also holding down the Ctrl key.
  • Zoom in and out by using the keyboard.  To zoom in, make sure that the graph has focus and press the Insert key or the + key. To zoom out, press the Delete key or the - key.

If no line or curve is displayed in the graphing area, then this may be because no part of the graph of the function is in the viewable area.  To reset the viewable area to what it was when the function was selected, click on the ‘Autoscale’ button.

7. Graph Options

To see the display options for the graph, click the Options tab. Each option can be turned on or off by clicking its checkbox. The options are as follows.

  • Crosshair replaces the cross cursor with a full crosshair extending to the edges of the graph.
  • Coordinates displays the coordinates of the cursor/crosshair position, next to the cursor or crosshair.
  • Grid displays grid lines corresponding to the scale markings. (The scale markings are chosen automatically.)
  • Axes displays the x- and y-axes.
  • Trace displays the coordinates of a point on the line or curve.  The point is determined by the cursor/crosshair position. It moves in discrete steps, so it is sometimes necessary to zoom in to reach the point required.
  • y-intercept displays the coordinates of the point where the graph of the function crosses the y-axis. The y-intercept is the y-coordinate of this point.
  • Radians changes the units for the x-coordinates from degrees to radians.  This option does not apply to functions defined using the custom function.

To hide the options, click the Functions tab.

8. Using the cursor/crosshair

When you move the mouse pointer over the graph, it changes to a cross-shaped cursor (or a full crosshair, if the Crosshair option is selected). The coordinates of the centre of the cross are shown in the input boxes labelled x and y at the bottom left of the right-hand panel.

You can set the cross position by typing coordinates into the x and y input boxes and pressing the Enter key. You can also move the cross by using the keyboard. To do this, first make sure that the graph has focus (indicated by a yellow box). Then you can move the cross slowly left and right by using the Home and End keys, or up and down by using the Page Up and Page Down keys. You can move the cross faster by also holding down the Ctrl key. Alternatively, you can move the cross by switching on Num Lock on your keyboard and using the arrow keys on the numeric keypad.

9. Finding the coordinates of a point on a graph

Sometimes you may want to find the coordinates of a point on a line or curve, such as an x-intercept, the vertex of a parabola or the intersection point of two graphs. You can find the coordinates of a point on a graph by using the Trace option and zooming in on the point.

The x-coordinate of a trace point is exact; the y-coordinate is calculated from this x-coordinate and is usually rounded.


To find the coordinates of a point on a graph

  • On the Options page, tick the Trace checkbox. It is usually helpful to untick the Coordinates and y-intercept checkboxes, to avoid having too many coordinates displayed on the graph.
  • Move the cursor left or right until the trace point is the point that you want. Its coordinates are displayed.
  • You may need to zoom in to obtain the exact point required, or to obtain enough precision. You can do this by pressing the ‘Zoom in’ button repeatedly, using the arrow buttons to keep the point in the viewable area, or by changing the x min, x max, y min and y max values.

To find the coordinates of an intersection point

  • As above, tick the Trace checkbox on the Options page.
  • If two graphs are drawn, a trace point appears on both graphs. Move the cursor left or right until the two trace points coincide at the intersection point.
  • You may need to change the viewable area so that the intersection point is in the middle, to ensure that both pairs of coordinates are displayed. You may also need to zoom in on the intersection point to obtain trace points that coincide, or to obtain enough precision.

10. Finding a value for y, given a value of x

  • On the Options page, tick the Trace box.
  • Enter the value for x into the box labelled x at the bottom left of the graph panel, and press Enter.
  • The point with this x-coordinate is plotted on the graph, and its coordinates are displayed. This gives the value of the y-coordinate.

Note that you cannot use this method to find a value of x given a value of y.

11. Custom function syntax

To enter your own function, choose ‘Custom function’ from the drop-down list of functions and enter the rule of the function in the custom function box, using the syntax described in this section.  The independent variable must be x.

For example, to enter the function with rule

y = x3 + 2.7 sin(πx) + e−x + 1,

enter the following expression in the box:

x^3+2.7*sin(pi*x)+e^(-x+1).

In general, the syntax is as follows.

  • * for multiplication
  • / for division
  • ^ for exponentiation
  • sqrt( ) for square root
  • sin( ), cos( ), tan( ) for sine, cosine and tangent
  • asin( ), acos( ), atan( ) for sin-1, cos-1 and tan-1(typing in sin^(-1), for example, gives a syntax error)
  • ln( ) for natural logarithm
  • e for the mathematical constant e = 2.718282
  • abs( ) for absolute value
  • max( , ) for the maximum of two values
  • min( , ) for the minimum of two values
  • floor( ), ceil( ), round( ) for the previous, next, and nearest integer, respectively

Here are some more examples of expressions that can be typed in the box:

e^(2-x)*ln(x^2+1)

tan(pi*x/3)+acos(x)

max(sin(2*x),0)

Note that the unit of measurement for custom trigonometric functions is the radian. If x is measured in degrees, then you will need to multiply it by pi/180 to change it into radians. For example, to plot y = sin x with x in degrees using the custom function option, enter sin (x*pi/180). Then alter the x min, x max, y min, y max values as appropriate.

If ‘Two graphs’ is selected, and one of your functions is a custom trigonometric function while the other is a predefined function, then you should make sure that the Radians option is ticked and that the predefined function also uses radians. If you do not do this, then the two graphs will be drawn with different units for x.

Take care with powers. For example, 2^3^4 means 2^(3^4) and sin(x+1)^2 means (sin(x+1))^2. When in doubt, use brackets!

You can omit a * between a number and a letter. For example, you can type 5x + 5sin(x) as an alternative to 5 * x + 5 * sin(x).

Custom functions with constants

A custom function can contain up to four constants, but these must be named A, B, C or D. For example, you could type any of the following expressions into the custom function input box:

A*x^2+B*x+C

A*x^3+B*x^2+3*x-2

A*(sin(B*x))^2+C*(cos(D*x))^2

An input box and a slider are provided for each constant.

12. Line thickness and colour options

The Options page provides two drop-down boxes to allow you to change the line thickness and the colours.

  • The options ‘Standard line thickness’, ‘Thicker lines’ and ‘Thickest lines’ allow you to change the thickness of the lines displayed in the graph, as well as the thickness and font size of the scale numbers on the graph and in the Information box.
  • The options ‘Standard colours’, ‘Black on white’, ‘White on black’, ‘Yellow on black’ and ‘Dark blue on light blue’ allow you to change the application's colours.

These settings are saved and remembered for when you next use the application.
